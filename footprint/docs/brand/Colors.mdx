# Color System

Footprint uses a vibrant yet refined color palette. The brand gradient (purple to pink) is the signature visual element, supported by accent colors and a clean neutral system for text, backgrounds, and borders.

All colors are defined as CSS custom properties in `globals.css` and extended in `tailwind.config`.

## Usage Guidelines

| Context | Color | Tailwind Class |
|---------|-------|----------------|
| Primary CTA buttons | Brand gradient | `bg-gradient-brand` |
| Secondary buttons | Brand purple | `bg-brand-purple` |
| Accent highlights | Brand pink | `text-brand-pink` |
| Info / links | Brand cyan | `text-brand-cyan` |
| Warnings / alerts | Brand orange | `text-brand-orange` |
| Body text | Text primary | `text-text-primary` |
| Captions / labels | Text secondary | `text-text-secondary` |
| Placeholders | Text muted | `text-text-muted` |
| Page background | BG white | `bg-light-bg` |
| Section background | BG light | `bg-light-soft` |
| Card background | BG soft | `bg-light-muted` |
