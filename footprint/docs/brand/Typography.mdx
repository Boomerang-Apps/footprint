# Typography

Footprint uses **Heebo** as its primary (and only) typeface. Heebo is a geometric sans-serif designed for both Hebrew and Latin scripts, making it ideal for our Hebrew-first, bilingual interface.

**Font stack:** `'Heebo', system-ui, sans-serif`

**Available weights:** 300 (Light), 400 (Regular), 500 (Medium), 600 (Semi-Bold), 700 (Bold)

## CSS Variables

```css
/* Font family */
font-family: 'Heebo', system-ui, sans-serif;

/* Weights */
font-weight: 300; /* Light — decorative, large display text */
font-weight: 400; /* Regular — body text, descriptions */
font-weight: 500; /* Medium — labels, nav items, h5/h6 */
font-weight: 600; /* Semi-Bold — h3/h4, buttons, emphasis */
font-weight: 700; /* Bold — h1/h2, hero text, brand name */
```
